# FEFFS

Application mobile fictive du **Festival Européen du Film Fantastique de Strasbourg (FEFFS)**, développée avec **Flutter** et compatible **Android & iOS**.

---

## Présentation

**FEFFS** est une application mobile permettant aux festivaliers de vivre le festival directement depuis leur smartphone. Elle offre une expérience autour de la programmation, des événements et des contenus quotidiens du festival.

---

## Fonctionnalités

*  **Scan de QR codes** via un lecteur intégré pour accéder rapidement aux contenus du festival
*  **Consultation du programme** du FEFFS
* **Ajout des événements en favoris**
* **Notifications** lors du clic sur un événement favori
* **Système de connexion utilisateur**
* **Achat d’un pass fictif** pour le festival
* **Accès à la quotidienne** du festival (actualités, temps forts, annonces)

---

## Technologies utilisées

* **Flutter** (Dart)
* **Android Studio**
* Application **cross-platform** : Android & iOS

---

## Démarrage du projet

### Prérequis

* Flutter SDK installé
* Android Studio ou VS Code
* Un émulateur Android / iOS ou un appareil physique

### Installation

```bash
# Cloner le dépôt
git clone https://github.com/votre-utilisateur/feffs.git

# Accéder au dossier du projet
cd feffs

# Installer les dépendances
flutter pub get

# Lancer l'application
flutter run
```
